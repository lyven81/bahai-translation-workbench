Anthropic API Key REQUIRED How do I get one?

Get your free Anthropic API key in 3 steps

  1. Go to console.anthropic.com/settings/keys and sign up with your email (free — takes 2 minutes).
  2. Click "Create Key". Give it any name (e.g., "Translation Workbench"). Copy the key that appears — it starts with sk-ant-...
  3. Paste the key into the box above and click Save Key. You're ready to translate.

Your key is stored only in your browser (localStorage) and sent directly to Anthropic when you run a translation. It is never stored on our server or shared with anyone. You can clear it any time.
